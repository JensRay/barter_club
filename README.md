## The Barter Club
A trading platform for bartering, money-free trading

http://www.the-barter.club/

A Rails app built as Le Wagon final project by Martin Nordlund, Jedrzej Lagodzinski, Alize Breezes and Anjte Heiermann.

Technologies:
- Ruby on Rails
- JavaScript
- CSS
- HTML
- Heroku
- API
